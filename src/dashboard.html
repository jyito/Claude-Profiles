Any network
To connect from outside your home network, set up Tailscale (free, no router config).
Requires Remote Login: System Settings → General → Sharing → Remote Login.
  1. On the Mac: turn on Remote Login (System Settings → General → Sharing).
  2. On your device: install an SSH app (Blink Shell, Termius).
  3. To reach it from any network: install Tailscale on the Mac and the device, sign into both with the same account, then use the "Any network" line above.
  4. Paste the line into your SSH app — you're in the session. Detach with Ctrl-A then D.
Full guide: docs/REMOTE.md in the repo.
